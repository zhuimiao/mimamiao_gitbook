# 密码喵
>密码喵是一个密码管理器。密码一定要专号专用，才能更有效的保护好自己的数据、财产安全。

## 对我来说的意义
从14年开始就从事iOS开发工作，一直在忙着做公司的项目，对公司项目对我来说是责任，我希望公司的项目越来越好。密码喵是我第一个个人项目，也是我的第一个swift项目， 是我的业余兴爱好。
